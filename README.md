# abrasive

## Development
We use [mise](https://mise.jdx.dev/) to manage toolchains and tasks. See the [mise install guide](https://mise.jdx.dev/installing-mise.html) to get set up.

### Marketing website
its an astro site hosted with the netlify free tier.
to run the dev server run this command: 
```sh
mise run website
```
To update the live site just merge a PR into the master branch that changes the site.
