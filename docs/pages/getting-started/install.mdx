# Installation

## From Source (Cargo)

Abrasive is written in Rust. Install it with Cargo:

```bash
cargo install abrasive
```

## From Release Binaries

Download pre-built binaries from the [GitHub Releases](https://github.com/Clavigers/abrasive/releases) page.

```bash
# Linux (x86_64)
curl -L https://github.com/Clavigers/abrasive/releases/latest/download/abrasive-linux-amd64 -o abrasive
chmod +x abrasive
sudo mv abrasive /usr/local/bin/
```

## Verify Installation

```bash
abrasive --version
```

You should see the installed version printed. Next, proceed to [Project Setup](/getting-started/setup).
